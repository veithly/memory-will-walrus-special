Memory Will

Six restrained SVG directions for a product that makes agent memory inheritance inspectable.

Will Ring

Selected direction. A durable ring surrounds two scoped memory lines and one successor seal.

Flatline Seal

The death beat sits inside a receipt shape. Strong for the demo moment, less flexible for app chrome.

Scoped Nodes

A predecessor, durable memory, and successor chain. Useful but closer to generic network symbolism.

Redaction Gate

Clear redaction metaphor. Good for proof pages, too literal for the core product mark.

Cold Restore

Document plus forward motion. Strong for iconography, but less distinctive as a brand mark.

Inheritance Stamp

Formal and receipt-like. Good secondary stamp for the success frame.
