## Contest
Sui Overflow 2026, `Special - Walrus`.

The track asks for functional AI agents or agent workflows that use Walrus as AI's verifiable data platform. In-scope work includes durable data/file access, shared context, portable memory, long-running workflows, multi-agent collaboration, artifact-driven workflows, and developer tools for Walrus or MemWal.

## Binding construct primer
The judge must be able to cause a Walrus- or MemWal-backed state change in the first minute, and then reopen, recall, restore, share, delegate, or inspect the resulting durable object or memory without treating Walrus as optional backstage storage.

What counts: Walrus or MemWal is on the critical path of agent state, memory, artifact, or verifiable data. The result can be reopened, recalled, restored, shared, delegated, or inspected through a blob id, Walrus object, memory space, trace, or Sui-linked proof.

What does not count: a generic chatbot, hidden server memory, optional upload target, static storage appendix, dashboard, proof packet, or slide-like multi-agent theater.

## Lane objective
Generate 6 to 8 weird but buildable concepts that could create a memorable room reaction while still being a real product and a legal Walrus track fit.

Prefer:
- a surprising first-minute transformation, simulation, challenge, duel, memory transfer, restore, or handoff;
- a visible agent pursuing a goal in a stateful micro-world;
- a judge-injected twist that forces replan and changes durable Walrus/MemWal state;
- a screenshot/share/replay artifact that feels unlike a normal AI app;
- a path to a serious recurring product after the spectacle.

Avoid:
- fantasy hardware, impossible integrations, pure games with no buyer, hallucinated protocols, or concepts that need months of backend work before the first demo;
- ideas that are wild only because the branding is strange;
- anything where Walrus could be swapped for localStorage without changing the consequence.
